    This project leverages Principal Component Analysis (PCA) and its variants—Standard PCA, Sparse PCA, and Kernel PCA—to classify brain tumors using image texture analysis. Instead of relying on traditional Convolutional Neural Networks (CNNs), the project applies the Gray Level Co-occurrence Matrix (GLCM) to extract texture features from brain tumor images. By reducing the dimensionality of these features with PCA, the project aims to improve classification accuracy and efficiency in distinguishing between glioma, meningioma, and pituitary tumors.
